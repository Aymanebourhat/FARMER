# UI Inspiration Folder

This folder contains AI Studio / Stitch generated UI references.

These files are NOT production frontend code.

Codex may use them for:
- layout inspiration
- spacing
- typography
- colors
- component structure
- visual direction
- responsive behavior

Codex must NOT:
- copy fake mock data into production
- copy fake API calls
- copy localStorage demo logic
- add unsupported routes
- add unsupported backend fields
- add package.json files from generated prototypes
- replace the real frontend architecture
- build features outside the current project phase

Production frontend code must be written inside:
- frontend/app/
- frontend/components/
- frontend/lib/
- frontend/types/

The backend API spec and project docs are the source of truth.

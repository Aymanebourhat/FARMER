# Out of Scope — V1

These features are explicitly excluded from V1:

| Feature | Reason |
|---|---|
| Government dashboard | Future enterprise layer |
| Online payments | Legal/fraud/refund complexity |
| Escrow | Requires marketplace trust and operational process first |
| Native mobile app | Web-first validates faster |
| QR animal tagging | Later verification phase |
| AI animal diagnosis | Medical/legal risk |
| Transport/logistics | Separate business problem |
| Livestock insurance | Regulatory complexity |
| Auctions | Not needed before basic listing liquidity |
| IoT sensors | Expensive distraction |
| Blockchain | No justified need |

If Codex starts adding any of this, stop it.
